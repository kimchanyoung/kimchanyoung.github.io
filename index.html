Portfolio

nghbor.ly

I worked as the team lead for nghbor.ly, a communal goods and services sharing app. Nghbor.ly is a platform on which users can 'ping' their neighbors when in need of immediate help, whether it's for some butter for cookies, or for help hanging a picture frame.

Nghbor.ly was built primarily on Ruby on Rails. MailChimp and Mandrill were used for instant e-mail notifications, and WebSocket and JQuery were leveraged to build the built-in chat system.

Zurb Foundation was implemented for a sleek, responsive, mobile-first experience.

The repo and further information can be found here.

Stack OverPhil

Stack OverPhil is a simple q&a style Stack Overflow clone built on Ruby on Rails. Best practices were practiced in implementing the questions, responses, answers, and votes. One new challenge for this project was handling polymorphic associations in Rails.

JQuery was implemented for selecting between the different view types for changing navigation types without reloading the page. No front end framework was used for this project.

The page was created largely as an exercise in writing clean and manageable code, as well as for practicing Agile project management methodology.

The repo and further information can be found here.
